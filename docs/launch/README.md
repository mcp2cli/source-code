## TL;DR — Launch Sequence

```
Week −2  Build audiences, prep all assets
Week −1  Soft launch to early access list, seed GitHub stars, publish docs
Day 0    Product Hunt drop @ 00:01 PST + Hacker News Show HN
Week 1   Content wave: YouTube tutorials, X threads, blog posts
Week 2   Community engagement, dev influencer outreach
Week 3   Medium/dev.to articles, video tutorials, integrations showcase  
Week 4   Retrospective, community roundup, Month 2 planning
```
